## 문제별 파일 규칙
각 문제 폴더 안에는 아래 파일들을 넣습니다:
- `solution.py` (또는 `.java`, `.cpp`)
- `README.md` ← 문제 링크 + 핵심 아이디어 + 시간복잡도 + 반례

### 예시: baekjoon/15649_N과M(1)/README.md
```markdown
# [15649번: N과 M (1)](https://www.acmicpc.net/problem/15649)

### 풀이 아이디어
- 전형적인 백트래킹 문제
- 방문 체크 배열 `visited[]` + 현재 경로 저장 리스트 사용

### 시간복잡도
- O(N!)

### 반례 / 실수했던 점
- 출력 형식을 `1 2 3`처럼 공백으로 구분해야 함

## 매일 커밋 규칙
- 항상 열심히
- 성공과 실패 이유 적기
